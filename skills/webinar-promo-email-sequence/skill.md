# 5-Day Webinar Promotion Email Sequence

Write a 5-email sequence that drives registrations for a free live webinar, using provided templates, examples, and event details.

## Step 1: Gather Inputs

You need 3 things before writing. Check what the user has provided:

**1. PDF with templates + examples**
If the user has attached a PDF, read it first. Extract:
- The fill-in-the-blank template structure for each of the 5 days
- Tone, phrasing patterns, and formatting from the examples
- Any structural elements specific to their style

If no PDF is attached, ask for it — or ask if they'd like you to proceed with a general best-practice structure.

**2. Webinar Details**

| Field | Status |
|-------|--------|
| Webinar Title | Required |
| Host Name | Required |
| Host Title/Credentials | Required |
| Date & Time | Required |
| Industry/Niche | Required |
| Core Topic or Theme | Required |
| Key Topics Covered (3-4 with short explanations) | Required |
| Main Value Proposition | Required |
| CTA Link | Required |
| Replay Availability | Required |
| Personal Story or Social Proof | Optional but recommended |
| Testimonials | Optional but recommended |

**3. Guidelines (pre-confirmed — apply to all emails):**
- Conversational and direct tone
- Subject lines: under 42 characters, sentence case
- Use bold and subheadings to break up text
- No emojis
- Bullets for readability
- CTA bolded in H1 format
- Always use numerals (5 not five)

If webinar details are missing, ask before writing.

## Step 2: Read the PDF Templates

If a PDF is attached:
1. Read the full PDF carefully
2. Identify the template structure for each of the 5 days (fill-in-the-blank sections)
3. Note the example emails — study their sentence rhythm, hook style, and how they transition to CTA
4. Use those templates as your scaffolding, filled in with the user's specific webinar details

The templates in the PDF take priority over the generic frameworks below.

## Step 3: Write the 5 Emails

If no PDF templates are provided, use these frameworks:

### Day 1 — The Announcement
*Goal: Generate awareness and first-wave registrations*
- Hook: Bold opening statement about the problem or outcome
- Briefly introduce the webinar — what it is, who it's for
- Name 2-3 specific things they'll learn (use bullets)
- State the date, time, and that it's free
- Single CTA to register

### Day 2 — The "Why This Matters" Email
*Goal: Agitate the problem; deepen relevance*
- Open by naming the struggle your audience knows well
- Connect that struggle to what the webinar solves
- Share a brief story (host's or customer's) that makes it real
- Remind them of the date/time
- Single CTA to register

### Day 3 — The Value Deep-Dive
*Goal: Overcome skepticism; show what they'll actually get*
- Break down what will be covered, topic by topic
- For each topic: name it + one-line benefit (use bullets)
- Add social proof if available (testimonial, result, or credential)
- Replay note if applicable (creates urgency either way)
- Single CTA to register

### Day 4 — The Social Proof / Urgency Email
*Goal: Push fence-sitters with proof and proximity*
- Lead with a testimonial or success story (before → after format)
- Remind them the webinar is [X days/hours] away
- Address one common objection ("Even if you're busy…", "Even if you've tried X before…")
- Replay scarcity if no replay: "This won't be recorded"
- Single CTA to register

### Day 5 — Last Chance (Day of Webinar)
*Goal: Final push — urgency, simplicity, brevity*
- Subject line: "It's today" or "Last chance — [Webinar Title]"
- Short email: 100-150 words max
- Remind them what they'll walk away with
- State the exact start time + timezone
- Single CTA (register or join link)

## Formatting Rules (Apply to Every Email)

**Subject lines**
- Under 42 characters
- Sentence case (capitalize first word only, plus proper nouns)
- No punctuation at the end unless a question
- No emojis

**Body**
- Short paragraphs: 1-3 sentences max
- Use **bold** for emphasis and key phrases
- Use subheadings (##) to break up longer emails
- Bullet points for any list of 3+ items
- Numerals always: 5, 10, 3-step, not five, ten, three-step

**CTA format**
```
# [Register now / Save your seat / Join us live] → [LINK]
```
- One CTA per email
- Bolded, H1 size
- Action-oriented verb

---

## Output Format

Deliver all 5 emails in sequence:

```
---
**Email [#] — [Day Label]**

**Subject:** [Under 42 chars, sentence case]

[Email body]

# [CTA Text] → [LINK]

---
```

After all 5 emails, include a **Placeholder Checklist** — a quick list of any fields still needing real information (e.g., `[INSERT TESTIMONIAL]`, `[CONFIRM REPLAY POLICY]`).
