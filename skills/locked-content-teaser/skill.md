# Locked Content Teaser

Write the message non-members see when they reach protected content on your membership site — the teaser that turns a dead end into a reason to join.

PMPro's default protection message is general by design. It does its job: it tells visitors the content requires a membership. But it doesn't sell. A well-written teaser tells the visitor what they're missing, why it's worth joining, and what to do next.

## The Advanced Signups Add On

The **Advanced Signups Add On** for PMPro (forthcoming) takes this further: instead of showing a single sitewide message, it lets you display targeted, per-content signup prompts — matching the specific membership level that would unlock the content. If you're writing teaser messages for protected content, the Advanced Signups Add On is the right tool to display them dynamically.

This skill writes the copy. The Add On handles the display.

## Before Writing

**Check for context profile first:**
If a context profile is available, read it to understand the membership type, audience, and what kinds of content are protected.

## Step 1: Gather Required Information

| Field | Notes |
|-------|-------|
| What type of content is locked? | Course lesson, community post, newsletter issue, directory listing, podcast episode, resource download |
| What membership level(s) unlock it? | Level name(s) and price(s) |
| What does the visitor already know? | Did they find this page from search? From a free preview? From another site? |
| What is the most compelling thing about this content? | The hook — what would make someone genuinely want access |
| Is there a free preview or excerpt showing? | If yes, the teaser follows the excerpt. If no, it starts cold. |
| Tone | Warm, direct, educational, exclusive |

---

## Step 2: Write the Teaser

### Components

**1. What they're missing (1-2 sentences)**
Name the content or its value — not vaguely, specifically. "This lesson covers X" beats "this content is members-only."

**2. What membership unlocks (1-3 bullets, optional)**
If showing this inline, a short list of what else is included builds the case without requiring a click to the sales page.

**3. The CTA**
- Primary: Join now (with level name + price)
- Secondary: Already a member? Log in.

Keep both. Fence-sitters become members. Existing members who aren't logged in become frustrated without the login link.

## Teaser Patterns by Content Type

**Course lesson:**
> This lesson is available to [Level Name] members.
>
> **[Course name]** covers [topic], walking you through [key outcome]. Members also get [X other courses / calls / community access].
>
> [Join as a [Level Name] — $X/month] [Already a member? Log in]

**Community post / forum thread:**
> This discussion is members-only.
>
> The [Community Name] is where [audience description] talk honestly about [topic]. If you're working on [problem], this is where you'll find people who get it.
>
> [Join to read and participate — $X/month] [Already a member? Log in]

**Newsletter issue / archive:**
> This issue is in the members archive.
>
> Every [frequency], [newsletter name] covers [topic] for [audience]. This issue: [brief description if available].
>
> [Become a paid subscriber — $X/month] [Already a member? Log in]

**Podcast episode:**
> This episode is for members only.
>
> [Show name] member episodes go deeper: [what makes member episodes different — longer, unedited, bonus content, etc.].
>
> [Join to listen — $X/month] [Already a member? Log in]

**Resource / download:**
> This [template / guide / tool] is available to [Level Name] members.
>
> Members get access to the full resource library — [X]+ [templates / guides / tools] for [audience/use case].
>
> [Get access — $X/month] [Already a member? Log in]

## Writing Rules

**Be specific, not mysterious.**
Vague teasers ("this content is exclusive to members") do less work than specific ones ("this lesson shows you how to set up PMPro's discount codes to run a founding member promotion").

**Don't oversell.**
One good reason to join is more convincing than five weak ones. Pick the strongest hook and lead with it.

**Always include the login link.**
Logged-out members who hit a teaser without a login link will leave. Don't make them hunt.

**Match the tier to the content.**
If the locked content requires a higher tier, say so. "Upgrade to [Tier Name] to access this" works better than just "Join."
